## HTML 中有哪些常见的实体字符？

HTML 中常见的实体字符：

|字符|HTML 实体代码|
|---|------------|
|`<`|`&lt;`|
|`>`|`&gt;`|
|`"`|`&quot;`|
|`'`|`&apos;`|
|`&`|`&amp;`|
|` ` (不换行空格)|`&nbsp;`|
|`©`|`&copy;`|
|`®`|`&reg;`|
|`™`|`&trade;`|
|`°`|`&deg;`|
|`¥`|`&yen;`|
